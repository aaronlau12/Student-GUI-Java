Student Database GUI
Overview

A JavaFX desktop application for managing student records stored in a Microsoft Access database. The application provides a graphical interface for searching, adding, updating, and deleting student records using JDBC.

Features
Search for students by ID
Insert new student records
Update existing student information
Delete student records
Input validation and error handling
JavaFX graphical user interface
Microsoft Access database integration

Technologies Used
Java
JavaFX
JDBC (UCanAccess)
Microsoft Access (.mdb)

Project Structure
StudentGUI.java – JavaFX graphical user interface
Student.java – Business object and database operations
RegistrationDB.mdb – Microsoft Access database

Running the Application
Install Java.
Add the UCanAccess JDBC library to the project.
Place RegistrationDB.mdb in the project folder (or update the database path in Student.java).
Run StudentGUI.java.

Skills Demonstrated
Object-Oriented Programming (OOP)
JavaFX GUI development
JDBC database connectivity
SQL CRUD operations (Create, Read, Update, Delete)
Exception handling
User input validation
